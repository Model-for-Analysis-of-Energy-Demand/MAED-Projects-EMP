## How to Contribute
To contribute a new project or update an existing one:
1. Select the the relevant country folder
2. Select the relevant EMP folder
3. Click on ‘Fork’
4. Select owner and name your forked repository
5. Click on 'Creat fork'
6. Click on ‘Add File’
7. Click on ‘Create New File’
8.	Name new folder ‘[name]/’
9.	Create a README inside the folder, describing the included MAED zip, presentation and report files.
10.	Click ‘Commit Changes’
11.	Add commit message and description
12.	Click ‘Commit Changes’
13.	Create a metadata.json with key details such as:
    - Country and sector(s)
    - File version (file_version)
    - MAED version (maed_version)
    - Time period
    - Authors/contributors
    - License
11.	Upload zip, presentation and report files (Add File > Upload files)
12.	Click ‘Contribute’
13.	Click ‘Open Pull Request’
14.	Add a title and description
15.	Click ‘Create Pull Request’
16.	Editor will review and approve pull request, once this has happened your content will be available on the main repository. If you want to make any edits to your files, please follow the same steps as before.
